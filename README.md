# Studio Sprint

Studio Sprint is a small React dashboard for planning creative focus sessions. It showcases how React Icons, React Toastify, and React Idle Timer can work together to provide clear feedback, track inactivity, and keep users in a productive flow.

## Features

- Session planner with goal, duration, and vibe inputs.
- Toast notifications for success, warning, and error states.
- Idle activity tracking that nudges the user when they stop interacting.

## Tech Stack

- React + Vite
- React Icons
- React Toastify
- React Idle Timer

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
